# Brainfuck-Interpreter
A BrainFuck interpreter written in C

### interpreter.c
An interpreter for "vanilla" Brainfuck

## interpreter_extended.c
A spinoff of Brainfuck I made.

New Syntax:
```
//SHORTHAND SYNTAX 

>>> == >3
++ == +2
---- == -4
<<<<< == <5

// FOR LOOP

{<2+3}5

Repeats <2+3 five times 
```
